Descrição do Projeto

Este projeto consiste em um relógio digital e um cronômetro desenvolvidos utilizando HTML, CSS e JavaScript. O relógio exibe a hora atual em formato digital, enquanto o cronômetro permite iniciar, pausar e parar a contagem de tempo.

Objetivo do Projeto

O objetivo deste projeto é proporcionar uma ferramenta simples e funcional para acompanhar o tempo. Além disso, serve como um exercício prático para aprimorar habilidades em desenvolvimento web.
